This is a restaurant website and it is ideal for local restaurant or takeway as they can take order easily and customer can also booked reservation from the website.
For managing the booking and reservation there is an admin dashboard and only admin can view this. print the order and confirm customer booking.More filter and optoins will be added later

Techs used to create the website is mainly nodejs, reactjs, postgresql, express js, drizzle for orm.

The website is reponsive and worked on laptop, tablet mobile.
